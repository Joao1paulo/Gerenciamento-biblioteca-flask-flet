Bem-vindo ao Sistema

Uma aplicação Full Stack construída com Flask e Flet.

Sobre o Projeto

Este sistema permite o gerenciamento completo do acervo de uma biblioteca, incluindo listagem, cadastro, edição e exclusão de exemplares. A arquitetura utiliza Flask no backend com documentação interativa via Swagger, e o framework Flet no frontend para garantir uma interface de usuário reativa.

Como Usar e Executar o Sistema

1. Iniciando o Backend (API)

  1. Abra o terminal e navegue até a pasta backend.
  2. Ative seu ambiente virtual.
  3. Execute o servidor rodando python app.py.
  4. O Swagger (Documentação da API) pode ser acessado e testado em:
    http://127.0.0.1:5000/apidocs

2. Iniciando o Frontend (Flet)

  1. Abra um novo terminal e acesse a pasta frontend.
  2. Ative o ambiente virtual desta pasta.
  3. Execute a aplicação rodando o comando flet run --web main.py.
  4. A interface será aberta em uma nova aba do seu navegador, conectada diretamente ao Backend. Nela você poderá visualizar os livros, adicionar novos, editar ou apagar.
