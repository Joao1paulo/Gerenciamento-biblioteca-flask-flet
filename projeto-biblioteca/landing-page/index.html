Gerenciador de Biblioteca

Aplicação Full Stack • Flask & Flet

Sobre o Projeto

Este projeto foi desenvolvido como avaliação da disciplina de LDW. O sistema permite o gerenciamento básico do acervo de uma biblioteca, oferecendo listagem e cadastro de novos exemplares. A interface foi desenhada visando simplicidade e facilidade de uso, seguindo um fluxo limpo.

Como Executar

  1. Backend: Navegue até a pasta backend, crie e ative uma venv. Instale as dependências com pip install -r requirements.txt e rode python app.py.
  2. Swagger: Acesse http://127.0.0.1:5000/apidocs no navegador para testar os endpoints da API.
  3. Frontend: Em um novo terminal, vá até a pasta frontend, crie outra venv, instale as dependências e inicie o app com flet run main.py.
